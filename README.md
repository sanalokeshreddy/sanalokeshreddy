### 🤖 [Gemma Web | Private AI Workspace](https://gemma-web-ai.vercel.app/)

<a href="https://gemma-web-ai.vercel.app/"><img src="https://img.shields.io/badge/🔗_Live_Demo-00C853?style=for-the-badge"/></a>
<img src="https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB"/>
<img src="https://img.shields.io/badge/WebAssembly-654FF0?style=for-the-badge&logo=webassembly&logoColor=white"/>
<img src="https://img.shields.io/badge/TensorFlow.js-FF6F00?style=for-the-badge&logo=tensorflow&logoColor=white"/>

**A fully private, browser-based AI workspace running Google's Gemma models entirely on-device.**

</div>

**✨ Key Features:**
- 🔒 **100% Private** - No data ever leaves your device
- ⚡ **Zero-Latency** - Runs directly in browser via WebAssembly
- 📄 **RAG Pipeline** - Context-aware conversations with your documents (PDF, TXT)
- 📱 **Offline PWA** - Full functionality without internet
- 💾 **Persistent Storage** - IndexedDB for models & conversation history

**🎯 Technical Highlights:**
- Engineered near zero-latency inference using WebAssembly and MediaPipe
- Implemented complete RAG pipeline with TensorFlow.js for embeddings
- Built robust client-side solution with IndexedDB and Web Workers
